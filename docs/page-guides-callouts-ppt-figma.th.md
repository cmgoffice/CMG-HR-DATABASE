# Handoff Callouts สำหรับ PowerPoint / Figma

เอกสารนี้จัดรูปแบบสำหรับคนทำสไลด์หรือวาง annotation บนภาพโดยตรง ให้ใช้ 1 บล็อกต่อ 1 ภาพ และ 1 บรรทัดต่อ 1 callout เพื่อหยิบไปวางต่อได้ง่าย

ขอบเขตเอกสารนี้มีเฉพาะ 3 หน้า:
- `HR Dashboard`
- `Project Dashboard`
- `ประเมินผลพนักงาน`

ภาพ `จัดการสิทธิ์การเข้าถึงตาม Role` ไม่รวมอยู่ในชุดนี้ตามขอบเขตงานปัจจุบัน
