# ระบบ Column Mapping แบบง่าย - เลือกแค่รหัสพนักงาน

## การเปลี่ยนแปลง

ปรับระบบ Column Mapping ให้ง่ายขึ้น โดย:
- ✅ **เลือกแค่คอลัมน์ "รหัสพนักงาน"** เท่านั้น
- ✅ **คอลัมน์อื่นๆ auto-map อัตโนมัติ** ตามชื่อที่ตรงกัน
- ✅ **ถ้ารหัสพนักงานไม่ซ้ำ → Import ทั้งหมดเลย**

## การใช้งาน

### ขั้นตอนที่ 1: เลือกไฟล์ CSV
คลิกปุ่ม "Import" และเลือกไฟล์ CSV

### ขั้นตอนที่ 2: เลือกคอลัมน์รหัสพนักงาน
Modal จะเปิดขึ้นพร้อม:
- 🔑 **Dropdown เลือกคอลัมน์รหัสพนักงาน** (จำเป็น)
- ✅ **แสดงคอลัมน์อื่นๆ ที่จะถูก auto-map**
- 📋 **ตัวอย่างคอลัมน์ทั้งหมดใน CSV**

### ขั้นตอนที่ 3: ยืนยันและ Import
- ระบบจะ auto-map คอลัมน์อื่นๆ ที่ชื่อตรงกัน
- เช็ครหัสพนักงานซ้ำ
- Import ข้อมูลทั้งหมด

## คุณสมบัติ

### 🎯 เลือกแค่รหัสพนักงาน
- แสดง dropdown ขนาดใหญ่ชัดเจน
- มี label "จำเป็น" สีแดง
- Auto-select ถ้าเจอคอลัมน์ที่ชื่อตรงกัน

### ✅ Auto-Mapping อัจฉริยะ
ระบบจะ auto-map คอลัมน์ที่:
- ชื่อตรงกันทุกประการ (case-insensitive)
- ตัวอย่าง: "ชื่อตัว" ← "ชื่อตัว"

### 📊 แสดงข้อมูลชัดเจน
- จำนวนคอลัมน์ที่จะถูก auto-map
- ตัวอย่าง mapping (แสดง 5 คอลัมน์แรก)
- แสดงคอลัมน์ทั้งหมดใน CSV พร้อมสีแยกประเภท:
  - 🔵 สีน้ำเงิน = รหัสพนักงาน (เลือกเอง)
  - 🟢 สีเขียว = auto-mapped
  - ⚪ สีเทา = ไม่ได้ map

### 🔍 ตรวจสอบรหัสพนักงานซ้ำ
- เช็คจากรหัสพนักงานในระบบ
- ไม่ใช่เช็คจาก Document ID
- แสดงรายการที่ข้าม (มีอยู่แล้ว)

## ข้อดี

✅ **ใช้งานง่าย** - เลือกแค่ 1 คอลัมน์
✅ **รวดเร็ว** - ไม่ต้องเลือกทีละคอลัมน์
✅ **อัจฉริยะ** - auto-map คอลัมน์อื่นๆ
✅ **ชัดเจน** - แสดงข้อมูลครบถ้วน
✅ **ปลอดภัย** - เช็ครหัสพนักงานซ้ำ

## การทำงานภายใน

### Auto-Detection รหัสพนักงาน
```typescript
const employeeIdHeader = csvHeaders.find(
  (header) => 
    header.trim().toLowerCase() === 'รหัสพนักงาน' ||
    header.trim().toLowerCase() === 'employee id' ||
    header.trim().toLowerCase() === 'employeeid' ||
    header.trim().toLowerCase().includes('รหัส')
);
```

### Auto-Mapping คอลัมน์อื่นๆ
```typescript
schemaFields.forEach((field) => {
  const exactMatch = csvHeaders.find(
    (header) => 
      header.trim().toLowerCase() === field.label.toLowerCase() ||
      header.trim().toLowerCase() === field.id.toLowerCase()
  );
  
  if (exactMatch) {
    mapping[field.id] = exactMatch;
  }
});
```

### Final Mapping
```typescript
const finalMapping = {
  ...autoMapping,           // คอลัมน์ที่ auto-map
  'รหัสพนักงาน': employeeIdColumn,  // คอลัมน์ที่เลือกเอง
};
```
